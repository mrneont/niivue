a NVImage encapsulates some images data and provides methods to query and operate on images
Parameters:
Name Type Attributes Default Description
dataBuffer array an array buffer of image data to load (there are also methods that abstract this more. See loadFromUrl, and loadFromFile)
name string <optional>
 '' a name for this image. Default is an empty string
colorMap string <optional>
 'gray' a color map to use. default is gray
opacity number <optional>
 1.0 the opacity for this image. default is 1
pairedImgData string <optional>
 null Allows loading formats where header and image are separate files (e.g. nifti.hdr, nifti.img)
cal_min number <optional>
 NaN minimum intensity for color brightness/contrast
cal_max number <optional>
 NaN maximum intensity for color brightness/contrast
trustCalMinMax boolean <optional>
 true whether or not to trust cal_min and cal_max from the nifti header (trusting results in faster loading)
percentileFrac number <optional>
 0.02 the percentile to use for setting the robust range of the display values (smart intensity setting for images with large ranges)
ignoreZeroVoxels boolean <optional>
 false whether or not to ignore zero voxels in setting the robust range of display values
visible boolean <optional>
 true whether or not this image is to be visible
useQFormNotSForm boolean <optional>
 true give precedence to QForm (Quaternion) or SForm (Matrix)
colorMapNegative string <optional>
 '' a color map to use for symmetrical negative intensities
frame4D number <optional>
 0 volume displayed, 0 indexed, must be less than nFrame4D
onColorMapChange function <optional>
 ()=>{} callback for color map change
onOpacityChange function <optional>
 ()=>{} callback for color map change
